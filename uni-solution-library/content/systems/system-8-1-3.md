# 8.1.3 安全教育管理子系统

## 系统概述
面向新工人入场、班前教育、专项作业教育和复训考核，将VR安全体验、教考一体机、二维码教育和无线WIFI教育统一纳入教育台账。

## 系统组成
- VR安全体验设备
- 教考一体机
- 二维码教育内容
- WIFI教育入口
- 题库、课程和学习记录

## 核心功能
- 课程学习、考试测评、错题记录和证书/合格状态管理
- 新工人入场三级教育、专项安全教育、班前教育
- 教育结果与门禁、实名制、班组管理联动
- 教育台账、统计报表和监管留档

## 系统亮点
- 把线下宣贯转为可记录、可统计、可追溯的数据
- VR体验提升高风险作业教育的沉浸感
- 可按客户项目制度配置课程和考试规则

## 应用价值
提升安全教育覆盖率和留痕完整性，支撑安全生产责任制落实。
