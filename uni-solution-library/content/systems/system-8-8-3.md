# 8.8.3 智能水电监测子系统

## 系统概述
通过智能水表、电表或采集终端对现场水电消耗进行实时监控，支撑节能降耗和成本分析。

## 系统组成
- 智能水表
- 智能电表
- 采集网关
- 能耗分析平台
- 异常告警规则

## 核心功能
- 水电用量实时采集、分区统计和趋势分析
- 异常用水用电、漏水漏电或过载提醒
- 日/月/项目维度报表输出
- 与绿色施工指标和经营分析关联

## 系统亮点
- 将四节一环保落到可量化数据
- 适合生活区、办公区、加工区分区计量
- 可支撑节能考核和成本核算

## 应用价值
减少能源浪费，提升绿色施工管理的量化水平。
