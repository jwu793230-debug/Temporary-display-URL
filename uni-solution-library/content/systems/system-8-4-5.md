# 8.4.5 龙门吊安全监测子系统

## 系统概述
面向梁场、加工场、材料堆场等龙门吊作业场景，对起重量、吊钩高度、行走位置、风速、防碰撞和视频画面进行监测。

## 系统组成
- 龙门吊安全监控终端
- 载荷/高度/行走/风速传感器
- 视频监控
- 驾驶室显示终端
- 平台分析模块

## 核心功能
- 起重量、吊钩高度、行走位置、速度和风速监测
- 超载、限位、偏载、防碰撞风险预警
- 视频画面、历史数据和报警记录查询
- 设备档案、维保、司机和作业记录管理

## 系统亮点
- 适合梁场和大型构件吊装场景
- 可与地磅、梁场监测和视频监控联动
- 支撑龙门吊运行安全数字化监管

## 应用价值
降低龙门吊超限、碰撞和违规作业风险，提升场站大型设备安全管理水平。
